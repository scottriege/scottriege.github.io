Introduction

On this website you will see a showcase of all the skills I learned this semester. To start is graphical design using Gimp. Where I learned the basics of color correction and making a collage of layered image The second is video analysis. Where I learned how to use the program Dartfish to do both movement analysis and how to tag footage for key moments and statistics and set up a tagging system. Ending with Excel, where I learned how to set up pivot tables with appropriate slicers, as well as how to write out custom formulas and color-coding data.
